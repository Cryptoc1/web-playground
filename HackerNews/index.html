About Hacker.news( )
Site Features / Current Limitations:

The default sources in the feed are /r/programming and /r/hacking. Users can add their own sources, but they are currently only limited to sub-reddits. Additionally, users can't yet remove sources from their feed without clearing the site localStorage (If you have Google Chrome, this is easy). Features we're hoping to come is the ability to add sources from sites like 4Chan, YoutTube and Twitter. If you have a site that you feel would fit well with this site, you can submit a suggestion here.

About the Developers:

This project was designed and coded by Samuel Steele, a member of Cryptocosm Developers. I (Sam) created this site because I wanted to play around with some APIs, particularly Reddit's. The site's design was greatly inspired Material Design, if it wasn't yet adherent.
